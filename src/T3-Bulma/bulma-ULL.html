Bulma is a Modern CSS framework based on Flexbox
In this page you can find the Corporate Brand Guidelines. There you can find the official colors and font
The official font for the ULL brand is Montserrat

The corporate typeface of the University of La Laguna is Argentum Sans It is an Open Font License typography so that it can be installed and used freely. The main weights recommended for an identifying use of the University are Semibold and Light.
Argentum Sans is an evolution of the Montserrat typography, so for Google documents Montserrat can be used as equivalent.
